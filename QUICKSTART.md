## 💡 Pro Tips

1. **Hot reload is your friend** - Edit and see changes instantly
2. **MDX = Markdown + React** - Use React components in docs
3. **Frontmatter matters** - Controls sidebar position, title, etc.
4. **Sidebar auto-generates** - Just create .md files, they appear!
5. **Dark mode built-in** - No setup needed

## 🚨 Common Questions

**Q: Where do I add new docs?**  
A: Create `.md` files in `docs/` folder. They auto-appear in sidebar!

**Q: How do I change colors?**  
A: Edit `src/css/custom.css` - change `--ifm-color-primary`

**Q: Can I use TypeScript?**  
A: Yes! Already set up. Create `.tsx` files in `src/pages/`

**Q: How do I add search?**  
A: Install search plugin or integrate Algolia (see `LEARNING_GUIDE.md`)

**Q: Where's the build output?**  
A: `npm run build` creates `build/` folder with static HTML
