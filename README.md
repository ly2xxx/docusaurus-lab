## 🛠️ Commands

| Command | Description |
|---------|-------------|
| `npm start` | Start dev server (http://localhost:3000) |
| `npm run build` | Build for production |
| `npm run serve` | Preview production build |
| `npm run deploy` | Deploy to GitHub Pages |
| `npm run clear` | Clear cache |

---

## 📁 Key Files

- **`docusaurus.config.ts`** - Main configuration
- **`sidebars.ts`** - Sidebar navigation
- **`docs/`** - Documentation pages
- **`blog/`** - Blog posts
- **`src/pages/`** - Custom pages
- **`static/`** - Static assets

---

## 💡 Experiments to Try

1. ✅ **Add new doc:** Create `docs/my-doc.md`
2. ✅ **Blog post:** Add markdown to `blog/`
3. ✅ **Custom page:** Create `src/pages/about.tsx`
4. ✅ **MDX features:** Use Tabs, Admonitions, Diagrams
5. ✅ **Theme colors:** Edit `src/css/custom.css`
6. ✅ **Versioning:** Run `npm run docusaurus docs:version 1.0`
